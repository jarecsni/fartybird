# Future Enhancements for Farty Bird

This document tracks ideas and features for post-v1.0 releases.

## Version 2.0 - Planned Features

### Character Skins
- **Farty Spaceman** - Space-themed character with rocket farts
- **Farty Eagle** - Majestic bird with powerful wing farts
- **Farty Dragon** - Fire-breathing (farting?) dragon variant
- Unlockable through achievements or in-app purchase
- Leverage modular asset system for easy swapping

### Power-Ups
- **Mega Fart** - Temporary super jump boost
- **Shield** - One-time collision protection
- **Slow Motion** - Brief time slowdown for precision navigation
- **Magnet** - Attracts score multipliers
- Power-ups appear randomly during gameplay

### New Obstacles
- **Flying Obstacles** - Moving hazards that patrol vertically
- **Rotating Obstacles** - Spinning barriers with timing challenges
- **Suction Zones** - Areas that pull the character down (require stronger farts)
- **Wind Tunnels** - Push character up or down

### Difficulty Enhancements
- **Hard Mode** - Faster scrolling, narrower gaps
- **Nightmare Mode** - Combination of all obstacle types
- **Progressive Difficulty** - Game gets harder as score increases
- **Daily Challenges** - Special obstacle patterns each day

### Social Features
- **Global Leaderboards** - Compare scores worldwide
- **Friend Challenges** - Challenge specific friends
- **Replay Sharing** - Share your best (or worst) runs
- **Achievements** - Unlock badges for milestones

### Gameplay Variations
- **Endless Mode** - Current gameplay (v1.0)
- **Time Attack** - Score as much as possible in 60 seconds
- **Survival Mode** - Limited farts, must use wisely
- **Zen Mode** - No obstacles, just relaxing flying

### Visual Enhancements
- **Parallax Backgrounds** - Multiple scrolling layers for depth
- **Weather Effects** - Rain, snow, wind particles
- **Day/Night Cycle** - Background changes over time
- **Animated Obstacles** - Pipes with steam, moving parts

### Audio Enhancements
- **Background Music** - Upbeat chiptune soundtrack
- **Multiple Fart Sounds** - Variety of comedic sound effects
- **Voice Lines** - Character reactions to events
- **Sound Packs** - Themed audio to match character skins

### Monetization (Optional)
- **Character Skin Packs** - Premium themed characters
- **Remove Ads** - One-time purchase
- **Power-Up Bundles** - Starter packs for new players
- **Cosmetic Effects** - Fart trails, particle effects

## Version 3.0 - Blue Sky Ideas

### Multiplayer
- **Race Mode** - Two players compete side-by-side
- **Co-op Mode** - Work together to navigate obstacles
- **Battle Mode** - Sabotage opponent with obstacles

### Level Editor
- **Custom Obstacle Patterns** - Create and share levels
- **Community Levels** - Play user-generated content
- **Level Rating System** - Vote on best community levels

### Story Mode
- **Campaign** - Series of themed levels with progression
- **Boss Battles** - Special challenge levels
- **Cutscenes** - Comic-style story between levels

### Cross-Platform
- **Android Version** - Expand to Google Play
- **Web Version** - Browser-based gameplay
- **Cloud Save** - Progress syncs across devices

## Implementation Notes

- All v2.0 features should work with the modular asset system
- Maintain 60fps performance target
- Keep core gameplay simple and accessible
- Test new features with small user groups first
- Prioritize features based on user feedback from v1.0

## User Feedback Tracking

Once v1.0 launches, track:
- Most requested features
- Common complaints or pain points
- Drop-off points in gameplay
- Average session length
- Retention metrics

Use this data to prioritize v2.0 development.
